All Commands I have used here


First I deploy my repo :
  1.   Click on create new repository
  2.  Come to command line, go to project directory and put:  $ git init
  3.   $ git add .
  4.   $ git commit -m "First commit".
  5.  At the top of your GitHub repository's Quick Setup page, click  to copy the remote repository URL.
  6.  In the Command prompt, add the URL for the remote repository where my local repository will be pushed.
  7.   git remote add origin remote repository URL
  8.  Sets the new remote: $git remote -v
  9.  Verifies the new remote URL
  10.   $ git push origin master

 Add new commit and push code on GitHub
  1.  Check status on git: $git status
  2.  Add all files want to commit using:  $git add [files name]
  3.  $ git commit -m "your commit message"
  4.  $git push origin master
